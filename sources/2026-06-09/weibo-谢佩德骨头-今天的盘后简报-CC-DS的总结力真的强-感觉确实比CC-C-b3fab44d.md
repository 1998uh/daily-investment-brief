### 2.3 风险三要素框架
冰冰在告别前留下了完整的分析框架——风险从单一市场因子逐步扩散至宏观因子：
#### 要素一：内部流动性挤压
- ZJH 对公募基金风格漂移、私募与量化的管制 → 减轻挤压效应但仍需时间
- 短期内仍有尾端效应，依旧是重要风险
#### 要素二：全球加息预期 → 资产全面承压
| 信号 | 状态 |
| ---- | ---- |
| 10年期美债收益率 | 再次突破 **4.5%**，逼近红线 |
| 美股融资加强 | 杠杆率高位 |
| 韩国股市 | 再次熔断 |
| 国内海外投资通道 | 已关闭（如富途事件） |
| 美联储政策 | 靴子未落地 |
> "加息预期下，体系三要素（流动性不利），使得各种资产承压。非农数据强劲 → 美国经济很好 → 不需要降息 → 加剧加息预期 → 引发更大的向下波动。"
#### 要素三：地缘与贸易风险共振
| 风险点 | 具体表现 |
| ---- | ---- |
| **土耳其** | 与以色列摩擦增多、抛售黄金储备、抛售美债、通胀加剧、里拉持续贬值、GDP下滑 → **可能是火药桶** |
| **日本** | 日元贬值加速、加息预期增强 → 日元-美元全球货币流通受影响 → 全球美元流动性危机 |

| **中日菲** | 关系恶化 → 二用物质禁令开始影响相关产业 |
| **美元指数** | 日欧贬值推动美元指数突破 **100** |
> "风险演绎的过程，从单一的市场因子，逐步扩散到宏观因子。市场领先于风险演绎下跌，充分计价加息预期下流动性危机对资产的折价。因此呈现出，无论什么板块都在下跌的既视感。"
### 2.4 警示与忠告
- **6月16日节点**：风险提前到来，但6/16并未过去，风险没有解除
- **世界杯**：球市对股市的流动性分化

## 五、🔍 机构视角与产业观察
### 5.1 大摩 Computex 闭门会核心纪要（via 陆家嘴幽灵）
#### CPU 叙事切换：从人机交互到 AI 调用
- 老黄说未来 CPU 和 PC 不是直接给人用的，而是被 AI 接管和调用
- Vera CPU 效率较 Intel X86 提升 **1.7-1.9 倍**，定价更高端（预计单颗 **>6000 美元**）
- 按今年 250-400 万颗出货量算，仅 Vera CPU 就能为英伟达贡献 **200 亿美元**收入
- AMD 通过在台电锁定 3nm 和 2nm 产能竞争，英特尔短期面临供给和良率挑战
#### AI PC：增量还是噱头？
> "大摩给出的判断相对保守。"
- 搭载 N1X 芯片的 AI PC 每季度出货量仅 20-25 万台
- 对联发科等厂商收入拉动非常有限（1-2%）
- 短期内 AI PC 还无法引发大规模消费者换机潮
#### 通信架构：光进铜退？
- 老黄原话："能用铜就尽量用铜，必要时才用光"
- 200Gbps 单通道是当前铜与光的分界线
- Rubin 架构在 Scale Up 上的需求提升倒逼 CPO 商用进度提前
- 目前面临光耦合等技术难点，各大封装厂全力攻坚
#### AI 硬件需求：不降反升
> "AI硬件需求不仅没有放缓，反而因为订单挤压和组件短缺，引发全产业链温和涨价。"
关键数据：
- 今年 CSP 出货量年增预计 **>30%**
- 不仅 AI 服务器供不应求，传统 X86 CPU 也出现紧缺
- 部分客户转向采购 Arm 架构 CPU 作为备选
- BMC 出货量预期从 2600 万大幅上调至 **3400 万颗**
- 客户已逐渐接受"芯片通胀时代"，不敢轻易砍单（重新排队要等到明年）
> "当前产业链特征：需求托底、供给受限、价格向上。大摩偏好具备高端品牌溢价、能顺利转嫁成本的大型企业。"
### 5.2 创新药与利率（via 陈达美股投资）
> "作为久期最长的股票型资产之一，创新药对利率高敏感。"
陈达用三张图论证 XBI（美股创新药 ETF）与利率的关系：
| 利率指标 | 与 XBI 关系 | 关键节点 |
| -------- | ----------- | -------- |
| **名义收益率** | 大部分时间负相关 | 2021年出现背离（通胀溢价干扰） |
| **实际收益率** | 压制更直接、更对称 | 2022年从-0.5%飙至2%+ → XBI暴跌>50% |
| **期限利差** | 顺周期特征 | 倒挂期 XBI 承压，当前约 0.38% 小幅正挂 |
> "实际收益率是剔除通胀后的真实资金成本。通胀因素会被企业产品定价能力部分抵消。实际收益率才是无风险利率的真爸爸。"
### 5.3 存储周期（via 陈达美股投资）
> "就算下一波产业浪潮如老黄所言是物理 AI 大爆发，存储还是坐在上面的而不是跪在下面的。确定性大。"
- 云侧 HBM / 端侧 DRAM / 云+端 NAND / 冷数据 HDD —— 物理 AI 胃口都大
- **但**：需求端确定性大概率已 price in
- **周期股的核心**：关注供给端比需求端重要得多（产能表、良率曲线、CAPEX 节奏）
- 真正赚钱的不是判断供给好坏，是判断**市场对供给的定价错在哪**
### 5.4 韩国总统喊话（via 淡淡的相思林）
> "韩国总统李在明周一表示：我认为韩国股市目前仍被低估，所有韩国人都从股市上涨中受益。"
淡淡的相思林评论：政治喊话在指数基金被动调仓面前效果有限。
