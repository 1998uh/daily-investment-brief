## 六、🟡 资源材料涨价链条：从氧化锆到六氟化钨，供给缺口有多大？

### 6.1 对日材料替代：一图看懂

| 材料 | 日企困境 | 国内机会 |
|------|---------|---------|
| 高端氧化钇 | 日企占七成以上产能 | 国产率不到两成，抢占份额 |
| 氮化铝商用粉 | 日企德山垄断70%+，已减产约30% | 国内供应商填补缺口 |
| 铝电解电容 | 日本尼吉康、佳美工宣布涨价 | 国内替代+涨价双驱动 |
| 钨棒 | 住友公告7月停售部分日本产制品 | 钨系材料国产化 |
| 光刻胶 | ArF/KrF国产化率<1%/3% | 极低基数+AI需求爆发 |

> “日企受制于原材料不足，产能受限，国内同行抢占市场份额。一图看懂氧化锆\氮化铝产业链。” ——suoluos

### 6.2 六氟化钨：被调研纪要引爆的“小材料大缺口”

| 指标 | 数据 |
|------|------|
| 全球名义产能 | 约9200吨 |
| 实际有效供给 | 约6700吨（美韩产能极少流通） |
| 2026年总需求 | 7500吨以上 |
| 静态缺口 | 至少800吨 |
| 国内价格 | 140-160万元/吨 |
| 海外价格 | 200-300万元/吨 |
| 未来2-3年新增产能 | 仅约1500吨 |

> “占全球供货核心的日企因长达3-4个月无法获取高纯钨粉面临产线停摆，产能预计会大幅削减，并且市场份额是永久性流失。” ——陆家嘴幽灵

> “六氟化钨在整体存储芯片制造成本中占比不足2%，下游厂商为保障供应链安全完全能够吸收溢价，预期今年三季度散单价格将触及全年顶峰。” ——陆家嘴幽灵
